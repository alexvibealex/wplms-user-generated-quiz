WPLMS User Generated Quiz adds functionality for mock quizzes in which user selects tags of questions .

== Description ==

This plugin adds functionality for mock quizzes in wplms .


Tutorial On how to setup and get started : <a href="https://vibethemes.com/documentation/wplms/knowledge-base/wplms-user-generated-quiz/">link</a>

= More Information =

Visit the <a href="https://vibethemes.com/">VibeThemes</a> for documentation, support, and information on getting involved in the project.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'WPLMS User Generated Quiz'
3. Activate WPLMS User Generated Quiz from your Plugins page. 

= From WordPress.org =

1. Download WPLMS User Generated Quiz.
2. Upload the 'wplms-user-generated-quiz' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate WPLMS User Generated Quiz from your Plugins page. 
= Once Activated =

1. After installing this plugin you can add shortcode in your quiz if you want to the quiz to be as mock quiz .
2. Use [user_dynamic_quiz questions="2" marks="2"] shortcode to mae users to generate 2 questions of 2 marks each from selected pool of question tags .
